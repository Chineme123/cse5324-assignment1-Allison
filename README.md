## Git Commands Used inThis Assignment

### Repository Setup

- `git clone `
- Clone repository from Github
- `git config --global user.name`
- Set Git username
- `git config --global user.email`
- Set Git email

### Basic Operations

- `git status`
- Check repository status
- `git add [file]`
- Stage files for commit
- `git commit -m "[mesage]"`
- Commit changes with message
- `git push origin [branch]`
- Push commits o remote repository
- `git pull origin [branch]`
- Pull changes from remote

### Branching

- `git branch`
- List all branches
- `git checkout -b [branch-name]`
- Create and switch to new branches
- `git checkout [branch-name]`
- Switch to existing branch
- `git merge [branch-name]`
- Merge branch into current branch

### Conflict Resolution

- Manually edited conflicted files
- Used `git add` to mark conflicts as resolved
- Commited merge with descriptive message

### Other Commands

- `git log --online --graph`
- View commit history graphically
- `git diff`
- View changes before committing
